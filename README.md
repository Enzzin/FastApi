Repudiation:
Não existe logs das ações realizadas permitindo que usuários neguem ter feito alterações ou exclusões
Mitigação:
Adicionar logging das operações (criação, edição, exclusão) com identificação do usuário e horário

Elevation of Privilege:
Não existe separação de permissões permitindo que qualquer usuário execute ações críticas como deletar ou atualizar tarefas
Mitigação:
Implementar controle de autorização com níveis de permissão (usuário e administrador)
